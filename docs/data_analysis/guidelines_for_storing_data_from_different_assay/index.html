    If you are not following the protocols below, ask Rasi first.

    Guidelines for all assays¶

    • Only raw data should be stored in /fh/fast/subramaniam_a/data/ folder. No processed files, summary tables or figures are allowed.
    • Every data containing raw data should have an associated sampleannotations.csv file. You should do this even for assays that yield a single image such as Western blots. The only exceptions are agarose gel images, Sanger sequencing data, and Nanodrop measurements (essentially only those measurements that are unlikely to be used in a publication or analyzed quantitatively).
    • Analysis for each experiment should be performed in a dedicated directory in /fh/scratch/delete90/subramaniam_a/user/USERNAME/git/GITHUB_REPOSITORY/.
    • Each analysis folder should contain a single README.md file at the top level. All other files should be stored in one of data, ,annotations, scripts, figures, tables subdirectories.
    • The data folder for an experiment should contain a sampleannotations.csv file.
    • All scripts in the scripts folder should be in the lab_analysis_code git repository.
    • Figures, tables, and annotation files should be included in the same lab_analysis_code repository, but do not include tables that are more than a few MB.
    • Tables should always be in .csv or .tsv format. Confirm that they are displayed natively (in pretty format) on Github.
    • Save figures in .pdf format or .png if the PDF is very large (more than a few MB).

    Agarose gel images¶

    • We usually do not need high-resolution images for analysis or publication, so save the image in .jpg format from the Gel-Doc computer.
    • The raw file should be stored in /fh/fast/subramaniam_a/data/USERNAME/gel_images/YYYYMMDD_expNN_short_description_of_experiment folder and filename should be YYYYMMDD_expNN_short_description_image.jpg.
    • You can annotate the jpg image using Inkscape. You can also invert the color and crop the image if necessary. You can then export it as PNG for uploading.
    • The annotated gel image should be uploaded to the project or your GitHub repo AND your Wordpress lab notebook.
    • *No files should be stored in the analysis folder /fh/fast/subramaniam_a/user/USERNAME/analysis/gel_images unless absolutely necessary.

    Flow cytometry¶

    • The raw .fcs files should be stored at /fh/fast/subramaniam_a/data/USERNAME/flow/YYYYMMDD_expNN_short_description_of_experiment without creating any subdirectory.
    • The above data directory should also contain a sampleannotations.csv file that provides annotations for each of the .fcs files following the convention in this file.

    Polysome profiling¶

    • The raw .csv files should be stored at /fh/fast/subramaniam_a/data/USERNAME/polysome_profiling/YYYYMMDD_expNN_short_description_of_experiment without creating any subdirectory.
    • The above data directory should also contain a sampleannotations.csv file that provides annotations for each of the .csv files following the convention in this file.
